# Resumo da Milestone M1

## Tema do Projeto
Manutenção preditiva de motores elétricos industriais com foco em redução de downtime não planejado.

## Problema Definido
Falhas inesperadas em equipamentos industriais causam interrupções na produção, aumento de custos e perda de eficiência operacional.

## Proposta
Utilizar dados operacionais e de sensores para identificar padrões que indiquem falhas ou anomalias antes que elas aconteçam.

## Objetivo da M1
Estruturar a base do projeto por meio de:
- definição clara do problema;
- organização do repositório;
- preparação do pipeline de ETL;
- documentação inicial da arquitetura de dados.

## Entregáveis da M1
- repositório GitHub estruturado;
- README documentado;
- requirements.txt criado;
- script inicial de ETL;
- documentação do fluxo de dados.

## Próximos Passos
- definir o dataset final;
- inserir dados brutos em `data/raw`;
- executar o ETL inicial;
- iniciar a análise exploratória na M2.
